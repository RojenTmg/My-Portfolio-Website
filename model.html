My Experience

The topic web development was completely new to me. I dedn't knew a single thing about it. But as I started knowing about it more and more day by day, it seemed quite interesting topic to explore. All the experience of me learning to code website in first term has been a wonderful journey from the start. I got to know about many new topics which i have never heard before.

There were three major topics which a web developer should learn. They are HTML, CSS and Java Script. But, our first term focused on HTML and CSS. Firstly, I got the idea of the working mechanism of the website on internet. I understood the basic requirement(coding) an which website are built in. On weekly basis, I covered many different topic areas such as HTML elements, CSS syntax, box model, chrome developer tools, web fonts, layout and positio, responsive web design, google material design, HTML forms and many more. Overall, my journey of learning web development has been great. Now, by knowing these basics rules and criterias of web development, I feel confident in building simple web pages.
